CSCI 250 [CSCI Elective] - Computer Organization Dec 18, 2021
The name of the class may be misleading, when joining this course I made the assumption that it would be file structures, proper techniques around file hierarchies and computer usage tips and tricks. Computer Organization was actually an assembly class in disguise. While we were taught about how computers work down to the bits, the majority of the work revolved around the assembly language. This class did involve one of my favorite projects that I've worked on, the Towers of Hanoi project. Where we had to use recursion and memory calls through Assembly. As someone who had never used assembly, when the project finally worked as expected with minimal instructions (aside from how it should work), I was shocked and thrilled at the same time. I was so proud of myself for creating this project on my own with processes and ideas we learned in class. This was, by no means, an easy class and I earned my second B due to the difficulty of computing concepts. One thing that has been very beneficial in future classes and personal projects is being able to convert binary to decimal or hexadecimal, and visa-versa.
